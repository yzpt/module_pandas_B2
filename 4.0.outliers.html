Outliers

Les outliers, qui s'cartent significativement de la norme, peuvent fausser les mesures de tendance centrale et influencer les analyses statistiques. Cet article explore les causes courantes des outliers, qu'il s'agisse d'erreurs ou d'introductions intentionnelles, et met en lumire leur importance dans l'identification des anomalies lors de l'analyse des donnes.

Qu'est-ce qu'un outlier ?

Un outlier est un lment ou un objet de donnes qui s'carte significativement des autres objets (considrs comme normaux). Identifier les outliers est crucial en statistique et en analyse de donnes, car ils peuvent avoir un impact considrable sur les rsultats des analyses statistiques. L'analyse visant dtecter les outliers est appele outlier mining (extraction des outliers).

Les outliers peuvent biaiser la moyenne (ou moyenne arithmtique) et affecter les mesures de tendance centrale, tout en influenant les rsultats des tests de significativit statistique.

Comment les outliers sont-ils causes ?

Les outliers peuvent tre causs par une varit de facteurs, rsultant souvent d'une variabilit inhrente aux donnes ou d'erreurs dans la collecte, la mesure ou l'enregistrement des donnes. Voici quelques causes courantes des outliers :

  • Erreurs de mesure : Des erreurs dans les processus de collecte ou de mesure des donnes peuvent entraner des outliers.
  • Erreurs d'chantillonnage : Dans certains cas, des problmes lis au processus d'chantillonnage peuvent engendrer des outliers.
  • Variabilit naturelle : Une variabilit inhrente certains phnomnes peut galement donner lieu des valeurs extrmes. Certains systmes peuvent prsenter des valeurs extrmes en raison de la nature du processus tudi.
  • Erreurs de saisie des donnes : Les erreurs humaines lors de la saisie des donnes peuvent introduire des outliers.
  • Erreurs exprimentales : Dans les contextes exprimentaux, des anomalies peuvent survenir en raison de facteurs incontrls, de dysfonctionnements d'quipement ou d'vnements inattendus.
  • chantillonnage de plusieurs populations : Les donnes provenant de populations multiples aux caractristiques diffrentes peuvent tre combines par inadvertance.
  • Outliers intentionnels : Des outliers peuvent tre introduits intentionnellement pour tester la robustesse des mthodes statistiques.

import seaborn as sns
import matplotlib.pyplot as plt


def removal_box_plot(df, column, threshold):
    sns.boxplot(df[column])
    plt.title(f'Original Box Plot of {column}')
    plt.show()

    removed_outliers = df[df[column] <= threshold]

    sns.boxplot(removed_outliers[column])
    plt.title(f'Box Plot without Outliers of {column}')
    plt.show()
    return removed_outliers


threshold_value = 0.12

no_outliers = removal_box_plot(df_diabetics, 'bmi', threshold_value)

Z-score

Le Z-score, galement appel score standard, est une valeur qui permet de comprendre quelle distance un point de donnes se trouve de la moyenne. En dfinissant une valeur seuil, on peut utiliser les Z-scores des points de donnes pour identifier les outlier.

Zscore = (valeur_donne - moyenne) / cart_type

Dans cet exemple, nous calculons les Z-scores pour la colonne age du DataFrame df_diabetics en utilisant la fonction zscore du module stats de SciPy. Le tableau rsultant z contient les valeurs absolues des Z-scores pour chaque point de donnes de la colonne age, indiquant de combien dcarts-types chaque valeur s'carte de la moyenne.

IQR (Inter-Quartile Range)

L'approche de l'IQR (Inter-Quartile Range) pour identifier les outlier est l'une des plus couramment utilises et les plus fiables dans le domaine de la recherche.

IQR = Quartile3 Quartile1

Dans cet exemple, nous calculons l'intervalle interquartile (IQR) pour la colonne bmi du DataFrame df_diabetics. La mthode commence par calculer le premier quartile (Q1) et le troisime quartile (Q3) en utilisant la mthode du point mdian, puis dtermine l'IQR comme la diffrence entre Q3 et Q1. Cela fournit une mesure de la dispersion des 50 % des donnes centrales dans la colonne bmi.

In[11]:
# np.percentile usage:
# np.percentile(array, n) returns the nth percentile of the array

# IQR
Q1 = np.percentile(df_diabetics['bmi'], 25, method='midpoint')
Q3 = np.percentile(df_diabetics['bmi'], 75, method='midpoint')
IQR = Q3 - Q1
print(IQR)

0.06520763046978838

Pour dfinir les valeurs limites des outlier, on tablit des bornes suprieures et infrieures en fonction de la plage normale du jeu de donnes. Ces bornes sont dfinies comme suit (on considre une valeur de 1.5*IQR) :

  • upper = Q3 + 1.5*IQR
  • lower = Q1 1.5*IQR

Dans la formule ci-dessus, selon les statistiques, une augmentation de 0.5 de l'IQR est prise en compte (new_IQR = IQR + 0.5*IQR) pour inclure toutes les donnes se situant dans 2.7 carts-types de la distribution gaussienne.

Suppression des outliers dans un jeu de donnes en utilisant l'IQR

Dans cet exemple, nous utilisons la mthode de l'intervalle interquartile (IQR) pour dtecter et supprimer les outlier dans la colonne bmi du jeu de donnes sur le diabte.

La mthode calcule les limites suprieure et infrieure bases sur l'IQR, identifie les indices des outlier l'aide de tableaux boolens, puis supprime les lignes correspondantes du DataFrame. Le rsultat est un nouveau DataFrame avec les outlier exclues. Les dimensions du DataFrame avant et aprs la suppression des outlier sont affiches pour comparaison.

In[13]:
# Importing
import sklearn
from sklearn.datasets import load_diabetes
import pandas as pd

# Load the dataset
diabetes = load_diabetes()

# Create the dataframe
column_name = diabetes.feature_names
df_diabetes = pd.DataFrame(diabetes.data)
df_diabetes .columns = column_name
df_diabetes .head()
print("Old Shape: ", df_diabetes.shape)

''' Detection '''
# IQR
# Calculate the upper and lower limits
Q1 = df_diabetes['bmi'].quantile(0.25)
Q3 = df_diabetes['bmi'].quantile(0.75)
IQR = Q3 - Q1
lower = Q1 - 1.5*IQR
upper = Q3 + 1.5*IQR

# Create arrays of Boolean values indicating the outlier rows
upper_array = np.where(df_diabetes['bmi'] >= upper)[0]
lower_array = np.where(df_diabetes['bmi'] <= lower)[0]

# Removing the outliers
df_diabetes.drop(index=upper_array, inplace=True)
df_diabetes.drop(index=lower_array, inplace=True)

# Print the new shape of the DataFrame
print("New Shape: ", df_diabetes.shape)

New Shape:  (439, 10)
